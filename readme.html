Advanced paper trading extension for meme coin terminals. Practice trading without financial risk on GMGN, Axiom, and other supported platforms.

How to Get Access

This extension is free but requires an activation code. To get your code:

  1. Sign up using our referral link: Get Access Here

  3. Receive your unique activation code via email
  4. Install the extension and enter your code

This helps us track usage and provide better support to our community.

Features

  • Multi-Platform Support: Works on GMGN.ai, Axiom.trade, and more
  • Real-Time Price Tracking: WebSocket integration for live price updates
  • Position Management: Track multiple positions across SOL and BNB chains
  • Performance Analytics: Detailed P&L tracking with calendar view
  • Quick Buy Buttons: Fast trading directly from token listings
  • Chart Integration: Entry/exit lines on TradingView charts
  • Export & Share: Share your trading performance

Installation

  1. Get your activation code (see "How to Get Access" above)
  2. Download the extension files
  3. Open Chrome and go to chrome://extensions/
  4. Enable "Developer mode"
  5. Click "Load unpacked" and select the extension folder
  6. Visit supported platforms and enter your activation code
  7. Start paper trading!
